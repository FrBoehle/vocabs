# Vocab Vista

Ein visuell attraktiver Sprachtrainer mit integriertem Wörterbuch und mehreren Lernmethoden.

## Features
- 📚 Wörterbuch-Ansicht mit Themenfilter.
- 🃏 Karteikarten-Modus mit echter Flip-Animation (Vorder-/Rückseite).
- 🖼️ Karteikarten können zwischen Text- und Bildansicht umgeschaltet werden.
- ⌨️ Schreibtraining zum aktiven Abrufen von Übersetzungen.
- ⚡ Schnell-Quiz mit Punkteanzeige.

## Lokal starten
```bash
python -m http.server 4173
```
Dann im Browser öffnen:

`http://localhost:4173`
