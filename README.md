# Vocab Vista (React)

Ein visuell attraktiver Sprachtrainer mit integriertem Wörterbuch und mehreren Lernmethoden – jetzt sauber in React-Komponenten aufgeteilt.

## Features
- 📚 Wörterbuch-Ansicht mit Themenfilter.
- 🃏 Karteikarten mit Flip-Animation, Bild/Text-Umschaltung und Streak-Highscore.
- 🔥 Satisfying Feedback: Konfetti, Erfolgs-Pulse, Shake bei Fehlern, Quiz-Feedbackzustände.
- ⌨️ Schreibtraining zum aktiven Abrufen von Übersetzungen.
- ⚡ Schnell-Quiz mit Punkteanzeige und deaktivierten Antworten nach Auswahl.

## Tech Stack
- React + Vite
- Component-basierte Struktur unter `src/components`
- Daten separat in `src/data`

## Setup
```bash
npm install
```

## Dev Server
```bash
npm run dev
```
